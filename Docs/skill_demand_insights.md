## Skill Demand Analysis (453 postings, Apr-Jun 2025)

### Top 5 overall skills
SQL (79%), Python (66%), Statistics (45%), Tableau (38%), R (37%)

### Key finding: DA vs DS skill divergence
- Data Analyst postings emphasize SQL + BI/visualization tools 
  (Tableau, Power BI, Excel, Looker)
- Data Scientist postings emphasize Python + ML/statistics 
  (Machine Learning, Statistics, R, TensorFlow)

### Practical implication
A fresher targeting Analyst roles should prioritize SQL fluency and one 
BI tool (Tableau or Power BI) over deep ML knowledge. A fresher targeting 
Scientist roles should prioritize Python and core ML/stats concepts over 
BI tool proficiency.
